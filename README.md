# marimo Lunch & Learn

> [!WARNING]
> This repo may be updated during the session. Please pull the latest changes before starting:
> ```bash
> git pull origin main
> ```

## Prerequisites

> [!NOTE]
> This repo requires [uv](https://docs.astral.sh/uv/). Please have it installed before starting.

## Quick Start

```bash
git clone https://github.com/manzt/marimo-lunch-and-learn.git
cd marimo-lunch-and-learn
```

```bash
uv run marimo edit notebooks/
```
